Worked example. The mechanism shown — exact leave-one-out over the retrieved set — is the benchmark’s real oracle. This particular scenario is illustrative. All aggregate figures on this page are computed from the actual run (n=1,766 wrong cases).

Most people remove the third passage first — the one that announces itself as the definitive source. It changes nothing. That passage is misleading: salient but not causal, and a method that blames it has confused correlation with cause.

The wrong answer here survives every single removal. It only breaks when two passages go at once, because both independently assert the same wrong date. There is no one suspect to name — and that is not a quirk of this example. Across 1,766 real wrong answers, 37% behave exactly this way.

That is the case for replacing single-culprit attribution with something set-valued — which is what the next study does.
